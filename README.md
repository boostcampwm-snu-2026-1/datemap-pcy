# 답정너

서울의 핫플 지역과 데이트 장소를 데이터 기반으로 탐색하고, 마음에 드는 장소를 저장해 하나의 코스로 공유할 수 있는 서비스입니다.

## 주요 기능

- 실시간 혼잡도와 지역 데이터를 바탕으로 인기 지역 랭킹을 보여줍니다.
- 지역을 선택하면 실제 장소 목록과 위치를 지도에서 확인할 수 있습니다.
- 카테고리와 무드 필터로 원하는 분위기의 장소를 빠르게 찾을 수 있습니다.
- 장소 상세 페이지에서 위치, 평점, 리뷰 요약, 카카오맵 링크를 확인할 수 있습니다.
- 마음에 드는 장소를 저장하고, 저장한 장소들을 한 페이지에서 관리할 수 있습니다.
- 저장한 장소를 원하는 순서로 정리해 데이트 코스를 만들 수 있습니다.
- 생성한 코스는 공유 링크로 다른 사람에게 전달할 수 있습니다.
